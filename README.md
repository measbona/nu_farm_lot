# NU Farm LoT

# How to build NU Farm LoT
# For IOS
+ Download & Install Xcode 
+ Inside Terminal, Run 
  ```sh 
  npm run open-ios
  ```
+ Double Click on `nu_farm_lot.xcworkspace`
+ Crt + B ( Build Debug )
# For Android
+ Download & Install Android Studio
+ Open Emulator in Android Studio
+ Inside Terminal, Run `npm run build-android-debug`
